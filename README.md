Chicken Invaders

این پروژه، پیاده‌سازی بازی  Chicken Invaders با استفاده از زبان Java و کتابخانه گرافیکی Swing/AWT است. بازی شامل ۸ مرحله با انواع دشمنان (Normal, Fast, Zigzag, Shooter)، غول‌های پایان‌مرحله، و سیستم ثبت رکورد (Leaderboard) می‌باشد.

  مشخصات دانشجو
 
نام و نام خانوادگی: علیرضا مرادی قشلاقی

 شماره دانشجویی: 40413023

  کنترل‌های بازی

* **حرکت به بالا:** کلید `Up Arrow` یا `W`
* **حرکت به پایین:** کلید `Down Arrow` یا `S`
* **حرکت به چپ:** کلید `Left Arrow` یا `A`
* **حرکت به راست:** کلید `Right Arrow` یا `D`
* **شلیک کردن:** کلید `Space`
* **توقف بازی (Pause):** کلید `P`
* **خروج از بازی / بازگشت به منو:** کلید `ESC`
  

 پایگاه داده (Database)
 
در این پروژه از پایگاه داده SQLite برای ذخیره اطلاعات کاربران و رکوردهای بازی استفاده شده است.
* مثلا database.db که فایل ذخیره اطلاعات است

 
 جداول اصلی:

    1. Users: شامل اطلاعات کاربری (Username, Password) و تنظیمات ذخیره‌شده (وضعیت صدای پس‌زمینه، شلیک و...).
    2. Records: ثبت امتیازات (Score) و بالاترین مرحله طی‌شده (Level) برای هر کاربر جهت نمایش در Leaderboard.

  راهنمای اجرا و وابستگی‌ها

فایل Chicken_invaders.jar را اجرا کنید و بازی اجرا میشود (در صورت نصب داشتن java بر روی سیستم)

نحوه کامپایل و اجرا:
1. ریپازیتوری را کلون کرده یا فایل ZIP را استخراج کنید.
2. پوشه `background`، `chicken`، `airplan` و `sound-effects` (شامل منابع تصویری و صوتی) حتماً باید در کنار فایل‌های اجرایی یا سورس‌کد قرار داشته باشند.
3. پروژه را در یک IDE (مانند IntelliJ IDEA یا Eclipse) باز کرده و کلاس `GameMain` را اجرا کنید.
